Perguntas Frequentes

Utilizamos as APIs para desenvolvedores dos provedores de LLM, garantindo que seus dados nunca sejam usados para treinamento. Também temos um banco de dados criptografado e acesso controlado.

Você continua usando o chat normalmente. Os tokens que passarem do limite do plano serão cobrados à parte, de acordo com o uso e o modelo de IA escolhido.

Atualmente suportamos os principais provedores, como: OpenAI (GPT-4), Anthropic (Claude), Google (Gemini) e estamos constantemente adicionando novos provedores conforme a demanda do mercado.

Nosso serviço utiliza os principais métodos de autenticação presentes no mercado — como por exemplo, o SSO da Microsoft.

Estamos trabalhando para oferecer integração com as principais ferramentas corporativas no mercado. Integrações personalizadas estão disponíveis para clientes corporativos.
